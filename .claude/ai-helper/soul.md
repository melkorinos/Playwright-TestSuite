# Soul — Project Identity

## Core purpose
Barebones QA skeleton for API and E2E testing using Playwright. Starting point for new QA projects — not a finished product, a launchpad.

## North star
"Every new QA project starts here and has a clean, consistent foundation from day one."

## Quality bar
- Tests are behavior-driven, not implementation-coupled
- API and E2E layers are cleanly separated
- No test bleeds state into another
- Playwright runs headless and fast in CI
